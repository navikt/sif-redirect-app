# sif-redirect-app

Enkelt redirect app for sykdom i familien

## Komme i gang

For å kjøre opp app-en i dev:

1.  `Lag .env filen i root: `.

REDIRECT_INGRESS=http://localhost:8080/familie/sykdom-i-familien
PUBLIC_PATH=
PORT=3001

2.  `npm i`.
3.  `npm run dev`.
